# php-control-practice

## 概要

COACHTECH教材のハンズオン演習で作成した、成績判定プログラムです。
点数に応じて評価を表示し、複数の学生の成績の一覧や、平均点などの統計情報を表示させています。制御構文を深く追求しながら作成しました。

## 使用技術

- PHP : 8.5.1

## 学んだこと

こちらも約2ヶ月の休学期間を経ての再挑戦でした。PHPや関数は苦手分野で、仮引数や変数に同じ英単語が入っていると混乱することがあったのですが、今後スムーズにコーディングできるよう時間をかけて理解を深めました。

- **出力制御の習得とフレームワークの有用性への理解**: `foreach` 文を用いて配列からデータを取り出し、要素の数だけ正確に繰り返し処理される仕組みを習得しました。さらに、`echo` の配置位置による出力タイミング（繰り返し内の即時出力とループ外の最終出力）の制御を学び、1つのファイルにロジックと表示を混在させる複雑さを経験しました。この実践を通じて、LaravelのBladeテンプレートのように「ロジックと表示（HTML）を分離して管理できる仕組み」がいかに優秀で、保守性の高い開発において不可欠であるかを改めて深く理解しました。
- **変数のスコープと条件分岐の理解**: 同じ変数名であっても、関数の外側（実引数）と内部（仮引数）では異なるスコープ（有効範囲）を持つ別個の変数であることを正しく理解しました。この仕組みを用いて値を安全に関数へ渡し、内部の `if/elseif` による条件分岐によって、点数に応じた評価が的確に出力されるデータフローを実装しました。
- **配列関数`count()`の活用**: 今回は`count()`を使用し、配列の要素数を数える処理をコーディングし、生徒の人数を算出しました。
- **変数の初期化と初期値設定の重要性**: データを集計（加算）する際は、まず空っぽの箱（`$sum = 0;` など）を用意しておかなければ、コンピュータは正しく中身を積み上げられないという、プログラムの根本的な仕組みを学びました。

## 動作確認（閲覧方法）

ローカル環境（PC内）でページを確認する場合は、以下の手順で行えます。

### 1.リポジトリのクローン

```bash
git clone git@github.com:gomashio-no-omusubi/php-control-practice.git
```

### 2. クローンしたディレクトリに移動

```bash
cd php-control-practice
```

### 3. 以下のコマンドを実行し、PHPの組み込みサーバーを起動

```bash
php -S localhost:8000
```

### 4. ブラウザでの表示

以下のURLにアクセスして動作を確認します。

http://localhost:8000/grade_calculator.php

_※確認終了後は、ターミナルで `Ctrl + C` を押してサーバーを停止してください。_
